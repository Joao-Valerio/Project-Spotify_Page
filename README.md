🎵 Spotify Page Clone
Uma réplica da interface inicial do Spotify, desenvolvida com React. Este projeto tem como objetivo recriar o layout visual do Spotify Web para fins educacionais e de prática em desenvolvimento front-end.

📸 Preview

🚀 Tecnologias Utilizadas
React

Vite ou [Create React App] (dependendo do projeto)

JavaScript

HTML5

CSS3

Font Awesome (ou outra biblioteca de ícones)

🛠️ Funcionalidades
Interface responsiva inspirada na homepage do Spotify

Barra lateral com menu de navegação

Área principal com destaques musicais

Player de música (visual apenas)

Integração de fontes e ícones personalizados

📦 Como Executar o Projeto
# Clone o repositório
git clone https://github.com/seu-usuario/spotify-page-clone.git

# Acesse o diretório do projeto
cd spotify-page-clone

# Instale as dependências
npm install

# Inicie o projeto
npm run dev
